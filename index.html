The Beep Boop Game!

The Beep Boop game will return a range of numbers from 0 to the user inputted number with the following exceptions:

  • Numbers that contain a 1: all digits are replaced with "Beep!"
  • Numbers that contain a 2: all digits are replaced with "Boop!"
  • Numbers that contain a 3: all digits are replaced with "I'm sorry, Dave. I'm afraid I can't do that."
